변수나 상수에 할당할 수 있는 값의 종류

  • 숫자 : number 정수 실수, NaN(not a number : 숫자가 아님)
  • 문자 : string = "",'',11 감싼 값들
  • 불리언 : boolean => true/false
  • 객체 : object => 1개 이상의 데이터가 특정한 구조를 가지고 있는 형태 {키 : 값} { } : 아무 것도 없는 사실도 1개의 객체로 취급
  • 배열 : Array => [1,2,3,...] 형태의 데이터 집합 (순서, 길이가 있음)
  • 알 수 없는 값 : null => 뭔가 있는 것 같긴한데 파악이 안 됨
  • 미지정된 값 : undefined => 값이 할당되지 않음
  • 데이터의 타입 확인 : typeof 변수

변수의 관리

  • 기본 자료형 : 값이 메모리의 주소 값이 가르키는 곳에 직접 할당되는 값
    넘버, 불리언, 스트링(참조자료형이지만 사용편의상 기본자료형과 같이 사용)
  • 참조자료형 : 값이 저장되러 있는 주소를 할당
    스트링, 오브젝트, 어레이, 펑션, 널, 언디파인드
  • == 같다 != 같지않다
    === 같다 (값과 타입)
    !== 같지않다 (값과 타입)

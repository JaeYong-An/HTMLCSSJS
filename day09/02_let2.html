자바 스크립트의 변수 사용

  • 값을 담고 있는 메모리 공간의 이름
  • 자바 스크립트에서 변수는 선언하는 역할만 함
  • var, let(ES6 부터는 let응 사용하는 것을 적극 권장) const
  • 자바와 달리 변수에 자료형을 표시하지 않음
  • let으로 선언하면 어떤 값이든 다 담을 수 있음
  • var name = 'hong'; var name = 'kim'; 가능
  • 자바에서의 지역변수 개념.
  • let 중복선언 안 됨, 재할당은 가능 let name = 'hong'; name = 'kim'
  • 스코프(scope : 사용 범위) { }에서만 사용 가능 스코프를 벗어나면 재선언 가능
  • 지역변수 스코프 : { }
  • const : 재선언 불가능, 재할당 불가능 const name = 'hong'; name = 'kim'; //불가능
  • 선언 이후 할당 고정 값을 가지는 경우 사용
  • 고정된 값을 사용하는 규칙, 참조, 룰, 객체, 엘리먼트 정의시 사용

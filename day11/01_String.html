가장 많이 다루는 데이터 타입 : String

  • java에서의 String메소드와 거의 유사함
  • 문자열 데이터는 index(0부터 시작)를 가지고 있음
  • 문자열 값 내에서 개행(br)하는 것을 추천하지 않는다
  • new String() : 객체 타입으로 선언 가능 / 추천하지 않음
  • leghth : 문자열의 총 길이
  • 추출관련 메소드 : substring(), subst(), slice()
  • 문자대체관련 메소드 : replace(old,new) => 첫 매칭되는 문자만 가능 / 대소문자 구분
  • trim() : 양쪽 공백 제거
  • padStart(길이, 채울 문자) : padEnd(길이, 채울 문자)
  • charAt(index) / indexof()
  • startWith("str") / endWith("str") 해당 문자열 앞 / 뒤에 str 유무 확인 return true / false

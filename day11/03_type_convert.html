데이터 타입을 원하는 형식으로 변경 (형변환, casting)

  • 문자 -> 숫자 : number("String"), parseInt("String"), parseFloat("String")
  • 숫자 -> 문자 : 123.toString(), (123.123)toString()
  • 문자 -> boolean : Boolean("true")
  • boolean -> 문자 : String(true)
  • number -> boolean : Boolean(number), Boolean(1), Boolean(0)
  • boolean -> number : Number(true), Number(false)

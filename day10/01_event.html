이벤트 객체는 전역 객체로써 모든 태그에 객체를 참조시킬 수 있다

  • a tag와 button tag에 이벤트를 주로 참조시켜 클릭에 대한 반응울 기대할 수 있게 만드는 게 일반적
  • a href 속성에 의해 자체 이동(동기화 방식)이 구현되어서 별도의 이동을 먹는 코드가 필요
  • 객체에 이벤트를 부여하는 방식에는 크게 3가지가 있따(각자 목적과 작동방식이 다르다)
  • 화면의 출력영역을 document 영역 (객체) 라고 한다
  • html태크는 html element, 속성은 attribute, 일반텍스트 textnote 라고 함
  • html로 출력하기 : innerHTML이라는 property를 사용
  • text로 출력 : innerText property 사용
  • html element를 id값으로 객체 가져오기 : getElementById(`idName`)
  • html element를 css 선택자를 사용하여 가져오기 : querySelector
